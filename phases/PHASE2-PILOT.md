# Phase 2 – Pilot Launch
**Timeline:** 16 Dec 2025 – 31 Jan 2026  
**Timezone:** Africa/Johannesburg

## Phase Outcome
First income generated; outpatient and aftercare pilots tested; data-tracking and billing live.

---

## Milestones

### P2-M1: Adult Outpatient Pilot
- **Owner:** Adult Clinical Lead
- **Due:** 01 Jan 2026
- **Status:** ⏳ PLANNED

**Description:**  
Run first 8-week adult OP group (max 9); track attendance, outcomes, billing accuracy.

**Expected Outcome:**  
Proof of concept; first medical-aid claims submitted.

**Checklist:**
- [ ] Recruit 6–9 participants for pilot cohort
- [ ] Confirm participant insurance/payment method
- [ ] Schedule 8-week programme (dates & times)
- [ ] Prepare clinical materials and assessments
- [ ] Set up attendance tracking system
- [ ] Conduct pre-programme assessments
- [ ] Deliver 8-week programme
- [ ] Track weekly attendance
- [ ] Conduct post-programme assessments
- [ ] Submit medical-aid claims for all sessions
- [ ] Document outcomes and participant feedback
- [ ] Calculate programme cost-to-serve

---

### P2-M2: 28-Day Admissions Optimisation
- **Owner:** Admissions Officer
- **Due:** 01 Jan 2026
- **Status:** ⏳ PLANNED

**Description:**  
Enforce deposit + pre-auth; daily occupancy/denial tracking; fix document gaps same day.

**Expected Outcome:**  
Stable utilisation; faster cash conversion on admissions.

**Checklist:**
- [ ] Implement deposit collection process
- [ ] Enforce pre-authorisation before admission
- [ ] Set up daily occupancy dashboard
- [ ] Track pre-auth denials daily
- [ ] Create same-day document resolution protocol
- [ ] Train admissions team on new process
- [ ] Monitor cash conversion cycle time
- [ ] Document successful and failed admissions
- [ ] Weekly review of utilisation trends
- [ ] Create corrective action plan for issues

---

### P2-M6: Pilot Review
- **Owner:** Project Manager
- **Due:** 31 Jan 2026
- **Status:** ⏳ PLANNED

**Expected Outcome:**

- ✅ Pilot review completed with validated model
